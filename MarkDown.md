# Инструкция по работе с MarkDown

## Заголовки
Для создания заголовков используется символ (#). Главный заголовок создается одним символом (#). Для заголовков нижнего уровня прибавляется по одному (#) на каждый уровень:  
## ## H-2
###  ### H-3
#### #### H-4

## Форматирование текста
Для **выделения текста жирным** используется парный символ (**) с обеих сторон текста. 
Для *выделения текста курсивом* используется символ (*) с обеих сторон текста.
